# vue-photonkit
Vue.js components for Photon

## Requirements
* Vue.js
* Photon

## Usage

```html
<script>
  import { Icon } from 'vue-photonkit'
  const { Icon } = require('vue-photonkit')
  // or
  import Icon from 'vue-photonkit/lib/Icon'
  const Icon = require('vue-photonkit/lib/Icon')
</script>
```

See [vue-play scenarios](./play)
